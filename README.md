## :computer: COMPILACIÓN Y EJECUCIÓN DE LOS PROGRAMAS

Para compilar el programa, teclea lo siguiente (es necesario tener el JDK):

```console
javac *NombreDelPrograma*.java
```

Y para ejecutarlo:
```console
java *NombreDelPrograma*
```
